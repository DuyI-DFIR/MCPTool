Installation guide.

1. Download the .zip file (https://github.com/wrrulos/MCPTool)

2. Extract the .zip file. (You have to have the main MCPTool folder)

3. Download Nmap (https://nmap.org/)

4. Download Python 3 (Includes PIP in the installation) (https://www.python.org/)

5. Download NodeJS (https://nodejs.org/es/)

6. Create an account on Ngrok (https://ngrok.com/)

7. Download Ngrok and link your account by putting your token in the terminal. (https://dashboard.ngrok.com/get-started/setup)

8. When your account is linked, move the ngrok file to the MCPTool folder

9. Open a terminal and navigate to the MCPTool folder. Then you need to run the following commands.

- npm install mineflayer
- npm install process
 
Windows:

- python -m pip install -r requirements.txt

Linux:

- python3 -m pip install -r requirements.txt

10. To open MCPTool you must use the command

Windows:

- python MCPTool.py

Linux:

- python3 MCPTool.py
